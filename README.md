# Full Stack HM Template

> Basic template for FS HM

## Backends

### Typescript + Express + TSOA

#### How to setup?

Just run:

```sh
yarn
```

#### How to run?

Just run:

```sh
yarn start:dev
```

### FastAPI

#### How to setup?

Just run:

```sh
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

#### How to run?

Just run:

```sh
uvicorn main:app --reload
```
